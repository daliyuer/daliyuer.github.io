免费搭建高速静态网站！白嫖 Google Firebase Hosting：每月 10GB 流量 + CDN加速！

发表于2025-06-01|更新于2025-11-13|实用教程
|总字数:791|阅读时长:2分钟|浏览量:

想要免费托管自己的个人网站、项目展示页或博客？Google Firebase Hosting 提供了一个非常适合开发者的解决方案：每月 10GB 流量、10GB 存储空间、全球 CDN 加速和 HTTPS 支持，统统免费！

什么是 Firebase Hosting？

Firebase Hosting 是由 Google Cloud 提供的静态网站托管服务，适用于部署 HTML、CSS、JS、图片等前端资源。它默认集成了 HTTPS、全球内容分发网络（CDN）和强大的构建工具，帮助你轻松上线自己的静态网站。

免费额度说明

Firebase 提供的 Spark 计划（即免费层）包含：

  • 每月 10GB 托管流量
  • 每月 10GB 存储空间
  • 全球高速 CDN + HTTPS 证书
  • 可自定义域名绑定
  • 不需绑定支付方式，项目不易被回收

官方价格页查看：Firebase Pricing

常见用途推荐

用途示例
个人博客搭配 Hugo、Hexo、Astro 等静态引擎
简历展示页前端开发者/独立程序员专属展示窗口
项目 Demo 展示前端作品、小工具、小网页合集
学习部署流程初学者练习 Git / CLI / Web 项目托管
公众号/站群引导页搭配自定义域名，用作中转页、推广页等

部署教程

推荐使用 Linux 环境进行部署，步骤如下：

第一步：创建 Firebase 项目

前往 Firebase 控制台 创建一个新项目。

  • 项目 ID 创建后不可修改，建议提前命名好（例如 myblog → 最终地址是 myblog.web.app）。
  • 可后续绑定自定义域名。

第二步：安装 Firebase 工具链

  1. 安装 Node.js LTS（包含 npm）：https://nodejs.org/ 下载 LTS 版本

  2. 打开终端安装 Firebase CLI：

  3. 登录授权 Firebase：


    复制授权链接至浏览器，登录 Google 账号，授权后返回命令行粘贴授权码。

第三步：初始化并部署网站

  1. 初始化项目：

    选择 “Use an existing project”，选择你刚刚创建的项目。

    设置部署目录为 public，选择不启用 GitHub 自动部署。

  2. 将你的网站静态文件放入 public 文件夹中。

  3. 部署上线：

    成功后控制台会显示你的网站访问地址（一般为：yourproject.web.app）。

也可以使用 Firebase 控制台绑定你自己的域名

注意事项

  • Firebase Hosting 只支持静态内容托管，不支持 PHP、Node.js 等动态后端。
  • 免费层不适用于高频访问的网站（如论坛、视频站），建议仅用于个人展示用途。
  • 虽然免费，但也不适合大规模商用或流量突增的项目
  • 每个账号每日建站次数有限，请合理使用资源
  • 项目无需绑定信用卡，无结算风险、长期稳定。

总结

Firebase Hosting 适合所有想要免费建站、部署前端项目、展示静态网页的人。
配合 Hugo / Hexo / Astro 等工具，可以轻松搭建一个支持 HTTPS、CDN 加速的专业网站，0 成本也能拥有属于你自己的个人主页！
